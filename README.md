# ALGII

Structured materials for Praktikum Allgemeine und Analytische Chemie II (ALGII) at RWTH Aachen University.

- BaSO₄ (CHEBI:133326) / SrSO₄ (CHEBI:747199) / CaSO₄ (CHEBI:31346)
- Al₂O₃ (CHEBI:30187) / Al(OH)₃ (CHEBI:33130)
- TiO₂ (CHEBI:32234)
- ZrO₂ (CHEBI:747200)
- Cr₂(SO₄)₃ (CHEBI:53471) / Cr₂O₃ (CHEBI:48242)
- Fe₂O₃ (CHEBI:50819)
- SnO₂ (CHEBI:52991)
- MnO₂ (CHEBI:136511)
